## Tagline
The OS for your agent's tools — search, browser, scrape, execute via reliable APIs

## Description
AI agents fail in production because their tools break — not because the LLM is wrong.

AgentForge gives your agents five reliable tools via clean REST APIs:
• Web search (no rate limits, no CAPTCHAs)
• Managed browser automation (Playwright + proxy rotation)
• Structured web scraping
• Document parsing (PDF, Word, HTML)
• Sandboxed code execution (Python, Node, bash)

One API key. Credit-based pricing. 100 free credits to start.

Built for LangChain, AutoGen, CrewAI, OpenAI Assistants, and custom agent frameworks.
